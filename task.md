## 需求
生成一个空项目，包含前端、后端。其中前端分为两个入口：用户端（H5）和管理端（PC）。本次仅生成项目结构与技术选型，不生成具体功能代码。
## 技术栈要求如下：
* 前端：
* 框架：React 18+ 配合 TypeScript
* 构建工具：Vite
* 路由：React Router v6
* UI 组件库：antd（PC端）、antd-mobile（H5端）
* 样式：LESS
* 后端：
* 运行时：Node.js 20+
* 框架：NestJS（原文中为 nest.js)
* 语言：TypeScript
* 数据库：Supabase
* 代码规范：ESLint + Prettier
* 交付产物视觉 / 交互是否且贴合场景，且设计美观
## 项目结构：
apps/ 
client/          # React 前端应用 
server/          # Node.js 后端应用
